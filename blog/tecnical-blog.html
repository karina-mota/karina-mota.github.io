Classes and ID Attributes

When you create a webpage with HTML, your page will initially look dull and uninteresting.
In order to improve the appearance of your webpage, you will need to use CSS codes and associate it with the HTML page you created.

CSS will tell the browser how to display the text and other content that you write in HTML.
This will allow you to use different fonts, colours, alignments, backgrounds, borders, forms, positioning, decorations and
many other items that will increase your viewerâ€™s interest to your webpage.

There are a few ways you can style your webpage using CSS.
You can change the styles using tag selectors or by using attributes.

In this blog we will discuss the use of attributes such as IDs and Classes and the difference between them.

Attributes helps us describe the contents of the page inside the HTML markup.

Class Attribute

If you want to add the same CSS styles on multiple HTML elements, you can use the class attribute.
Classes can be added to multiples HTML elements and html elements can have multiple classes.

You will reference your class attribute by putting a . in front of their names.

ID Attribute

ID should be used sparingly. They are unique. They are not reusable.
HTML elements can only have one ID and IDs can only be used once per page.

ID attributes will override Class attribute.

You will always reference IDs by putting a # in front of their names.
